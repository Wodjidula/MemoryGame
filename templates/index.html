Instructions du Jeu

Bienvenue dans le Jeu de Mémoire ! Voici comment jouer :

  • Au début de chaque niveau, un ensemble de cartes est mélangé et affiché face cachée.
  • Votre objectif est de cliquer sur les cartes dans l'ordre croissant (1, 2, 3, etc.).
  • Si vous sélectionnez une carte incorrecte, toutes les cartes seront réinitialisées.
  • Terminez le niveau le plus rapidement possible pour atteindre un score élevé !
  • Utilisez le bouton "Niveau suivant" pour progresser ou "Recommencer" pour repartir à zéro.

Amusez-vous bien et essayez d'atteindre le niveau le plus élevé possible !
